# Starter Reset

## Program Description

Soft reset for a shiny starter.

<img src="images/StarterReset-0.png">

## Switch Settings

1. Full screen/match the image above
2. No filter applied

## Game Settings

1. Text Speed: Fast
2. Frame: Type 1

## Instructions

1. Stand facing the Pokeball of the starter you want.
2. Save the game. Return the menu cursor to the top after.
3. Start the program in-game.

## Options

### Go Home when Done:

Go to the Switch Home to idle when finished.
